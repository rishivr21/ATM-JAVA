# ATM Simulation in Java

This project is a simple ATM simulation written in Java.  
It demonstrates basic object-oriented programming concepts such as classes, methods, and user input handling.


## Features
- PIN authentication
- Check account balance
- Withdraw money
- Deposit money
- Exit option
